Common Licence Shaker

Download this project as a .zip file Download this project as a tar.gz file

Ce projet est piloté par l'Ecole de Droit de la Sorbonne (Paris 1) et mené avec le DILAb, le M2 Droit du Numérique de l'Univeristé Paris 1 et le DU Droit & Informatique.

L'ambition est de développer une aide à la décision dans le choix des licences. L'arbre décisionnel se fonde pour l'instant sur les principales licences existantes.

Présentation du prototype de générateur de licences pour le séminaire international de l'ENSV from Thomas Saint-Aubin on Vimeo.

N'hésitez pas à forker, le code est sous MIT Licence.
